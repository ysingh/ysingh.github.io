## Binary
- Two digits
- 0 or 1
- n binary digits can represent 2^n values. Eg: 1 binary digit can represent 2^1 values = 2 values ( 0, 1). 2 binary digits can represent 2^2 = 4 values (00, 01, 10, 11)(decimal values 0, 1, 2, 3).
